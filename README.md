# TorqueDen

> Where the build lives.

TorqueDen is a social community app for car enthusiasts — the digital garage-clubhouse
where you show your build, log your mods, swap specs and advice, and find your crew.
Each **car** is the profile, and its history follows it across owners.

Built with **Flutter** (Android · iOS · web) and **Supabase** (planned backend).

## Brand & theme

See [`TorqueDen-brand-and-theme.md`](TorqueDen-brand-and-theme.md) for the full brand guide —
palette, typography, voice, and components. The app theme in `lib/main.dart` is derived from it.

## Getting started

This is a standard Flutter project.

```bash
flutter pub get        # install dependencies
flutter run -d chrome  # run in the browser
flutter run            # run on a connected Android device / emulator
```

New to Flutter? Start with the [online documentation](https://docs.flutter.dev/).
